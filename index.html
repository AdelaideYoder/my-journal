Tuesday, May 22, 2018

Today was an interesting day. We started by creating a new repository on GitHub. There is a lot to remember for this. Before you start doing anything in Visual Studio Code, you have to go to your Terminal and make sure you are in your Workspace so that you can then create a new directory.

  1. Create New Repository on GitHub
  2. Go to Terminal
  3. Make sure you are in your Workspace Directory
  4. Create a new directory (example: mkdir my-journal)
  5. Then cd into your new directory
  6. At this point you could go ahead and do 'git init'
  7. Then create your main files using the 'touch' command
  8. DO NOT FORGET your README.md file
  9. Files you need: README.md, index.html, main.css, main.js
  10. Using 'git status' is helpful to make sure you are on the right path
  11. Then use 'git add' to add all of your new files
  12. Then use the 'code .' command to open up all of these in Visual Studio Code
  13. Now that you are in Visual Studio Code editor, open up your index.html file
  14. Use the '! (bang) - tab' to add in the HTML
  15. Then connect your main.css and main.js files to your HTML file
  16. NOTE: The CSS file needs to be added in the HEAD tag
  17. NOTE: The JavaScript file needs to be added in a Script Tag at the end of your HTML file right before the closing body tag
